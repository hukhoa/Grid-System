Guide to the Grid System

This guide explains how to use the grid system defined in grid.css.

1. Grid Container

To create a grid, wrap your content inside a container with class .grid.

<div class="grid"> ... </div>

2. Rows and Columns

Inside the grid container, use .row to define rows. Each row contains columns using the .col class. 

<div class="row">
    <div class="col c-4">Content</div>
    <div class="col c-4">Content</div>
    <div class="col c-4">Content</div>
</div>

3. Column Sizing

Columns are divided into a 12-part grid. Example:

  • c-6: Takes up half of the row (6/12 columns).
  • c-4: Takes up one-third of the row (4/12 columns).
  • c-3: Takes up one-fourth of the row (3/12 columns).

4. Offset Columns

Use c-o-* to shift columns to the right.

<div class="col c-4 c-o-2">Shifted Column</div>

5. Responsive Design

You can use m-* (medium screens) and s-* (small screens) for responsive layouts.

<div class="col c-12 m-6 s-4">Responsive Column</div>
